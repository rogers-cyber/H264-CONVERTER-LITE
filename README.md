# H264 CONVERTER LITE – Lightweight Raw H264 Video Converter v2.0.0

H264 CONVERTER LITE v2.0.0 is a lightweight desktop application designed to quickly convert raw H.264 video streams into modern video formats using a fast, simple, and fully offline workflow. It supports batch processing, drag-and-drop importing, real-time progress tracking, and reliable FFmpeg-powered encoding.

Built with Python, Tkinter, ttkbootstrap, and tkinterdnd2, H264 CONVERTER LITE is designed for CCTV workflows, surveillance footage recovery, developers, video technicians, and users who need a dependable offline H.264 conversion utility without unnecessary complexity.

USAGE GUIDE
------------------------------------------------------------

1. Import H264 Files  
Drag and drop .264 or .h264 files into the application or click "Select H264 Files".

2. Select Output Folder  
Choose where converted videos will be saved.

3. Configure Export Settings  
Choose output format and optional filename prefix.

4. Start Conversion  
Click "Start Conversion" to begin processing files.

5. Monitor Progress  
View real-time logs and progress updates.

6. Access Output Files  
Optionally auto-open the output folder after completion.

------------------------------------------------------------
CONVERSION WORKFLOW
------------------------------------------------------------

1. User imports raw H264 files  
2. Application validates supported inputs  
3. Output folder is selected  
4. FFmpeg initializes raw H264 decoding  
5. Selected encoder processes output format  
6. Files are saved using safe filenames  
7. Progress bar updates in real time  
8. Logs display conversion activity  
9. Completion summary is displayed
